Quiz App / Spurningar próf.

Það sem ég notaði:

npx create-react-app simple-quiz
cd simple-quiz

Bætti við Tailwind CSS:
npm install -D tailwindcss
npx tailwindcss init

lagaði tailwind.config.js til og bætti í index.css
@tailwind base;
@tailwind components;
@tailwind utilities;
